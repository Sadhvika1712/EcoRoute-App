# EcoRoute - Sustainable Navigation & Carbon Tracker

## Description
A web application for calculating carbon emissions based on real-time routes between cities like Sangareddy and Hyderabad. EcoRoute helps users make eco-friendly travel decisions by comparing the carbon footprint of different transport modes (Car, Bus, Bike) and illustrating the real-world impact of those choices.

## Features
- **Real-time Routing & Geocoding**: Search for start and destination locations using the OpenStreetMap Nominatim API.
- **Map Integration**: Beautiful and smooth interactive map visuals integrated with Leaflet.js.
- **Sustainability Impact Panel**: Live calculation of your trip's "Eco-Score", CO₂ savings compared to driving, and the equivalent number of mature trees needed to offset the emissions.
- **PDF Report Generation**: Generate, format, and download a professional trip report instantly summarizing your carbon footprint using jsPDF.

## How to Use
1. Clone the repository and navigate into the project directory.
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Start the development server by running:
   ```bash
   npm run dev
   ```
4. Open the local link provided in your terminal (usually `http://localhost:5173/`) to view and interact with the application. Enter locations (e.g., "Sangareddy" and "Hyderabad") to calculate your green route!
